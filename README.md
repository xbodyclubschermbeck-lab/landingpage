# EMS Landingpage

Eine moderne Landingpage für ein EMS-Studio zur Anfrage eines kostenlosen Probetrainings.

## Inhalt

Die Seite enthält:

- eine Hero-Sektion mit klarem Call-to-Action
- Informationen zu den Vorteilen von EMS-Training
- ein Kontaktformular für Probetrainings
- Felder für:
  - Vorname
  - Nachname
  - Telefonnummer
  - E-Mail-Adresse
  - bevorzugte Kontaktzeit

## Ziel

Diese Landingpage dient dazu, neue Interessenten für ein EMS-Probetraining zu gewinnen und einfache Kontaktanfragen online zu sammeln.

## Dateiübersicht

- `index.html` – komplette Landingpage

## Nutzung

1. Datei `index.html` herunterladen oder lokal speichern
2. Im Browser öffnen zum Testen
3. Auf GitHub hochladen
4. Optional über GitHub Pages, Netlify oder Cloudflare Pages veröffentlichen

## Hinweise

- Der Studio-Name kann im Code individuell angepasst werden
- Impressum und Datenschutz müssen mit echten Angaben ergänzt werden
- Das Formular ist aktuell für statisches Hosting vorbereitet
